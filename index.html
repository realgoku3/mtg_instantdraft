Instant Draft Machine

Generate authentic booster packs from any set

How to Use

Set Draft — Enter a set code (e.g. dsk, lci, mh3) and the number of packs to open. The machine builds a full card pool from Scryfall and rolls authentic boosters with correct rarity distributions, bonus sheets, and special slots.

Chaos Draft — Each pack comes from a different random draftable set. Great for wild, unpredictable games.

Block Draft — Pick a historical block and a number of drafters. Each drafter gets 3 packs, one from each set in block order, just like the old days.

Cube Draft — Paste a CubeCobra cube ID or URL, or paste a card list directly. Packs are drawn randomly from the list.

Quick Print — Connect a thermal receipt printer via WebUSB and print any card. Enter a card name (e.g. Lightning Bolt), a set+collector number (e.g. stx102), or prefix with a number for multiple copies (e.g. 4 Lightning Bolt, 4 stx102). Works with most ESC/POS printers (Chrome or Edge required).

Print a List — Paste a list of cards in the format [qty] [name] ([set]) [collector#] to batch-print multiple cards at once.

Set codes can be found at scryfall.com/sets.
